# Drawer
Shows a drawer that slides in and out from the right side of the screen.

May add directions soon.

## Props
| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `open` | _boolean_ | | If `true`, the drawer will be open

## Usage
```html
<script>
    let open = false
</script>

<Drawer {open}>
    Drawer Content
</Drawer>
```
