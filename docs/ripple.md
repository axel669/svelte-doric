# Ripple
> svelte-doric

Adds a ripple effect to an element.

## Props
This component takes no props.

## Usage
The element that should have the `Ripple` effect must be `position: relative` in CSS.
```html
<style>
    container-element {
        position: relative;
    }
</style>

<container-element>
    <Ripple />
</container-element>
```
